# pydoit: An API Wrapper around the i-doit API

## Getting started

### Installation

Install pydoit using pip:

```txt
pip install pydoit
```

### Example Code

```python
from pydoit import Idoit
```

## Currently supported methods

```txt
- idoit.login
- idoit.logout
- idoit.search
- idoit.version
- idoit.constants

- cmdb.object.create
- cmdb.object.read
- cmbd.object.update
- cmdb.object.delete
- cmdb.object.recycle
- cmdb.object.archive
- cmdb.object.purge
- cmdb.object.markAsTemplate
```
